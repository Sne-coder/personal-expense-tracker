# Personal Expense Tracker

A Python-based expense tracking application that helps users manage daily spending and monitor monthly budgets.

## Features

* Add new expenses
* View all expenses
* Filter expenses by category
* Monthly budget tracking
* Expense summary reports
* CSV file storage

## Technologies Used

* Python
* CSV Module
* Datetime Module
